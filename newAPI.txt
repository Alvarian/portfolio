worker functions
================
function getReference
	switch for Site/Service/Script
		case Site 
			look for url in item.url
		case Service
			fetch readme and split it for list of image urls and instruction
		case Script 
			fetch dedicated file or readme that contains logic file url

fetch list of repos from github
	for each item get description
		split description based on "|"
		have index of 2 from split description equal to presentable
		have index of 1 from split description equal to project type
		
		if presentable equals ":heavy_check_mark:" return getReference(projectType, item)
